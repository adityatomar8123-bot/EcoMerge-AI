# Legacy Notes

The repository previously included research-oriented ESG document tooling and generated artifacts. Those files have been removed from the active project surface so the hackathon submission can focus on EcoSphere.

## What Changed

- The frontend was moved to `frontend/`.
- Old report-demo UI components, generated outputs, logs, uploaded files, and temporary artifacts were removed.
- Backend startup now targets `app.main:app`.
- Dependencies were reduced to the local FastAPI and ERP foundation.

## Current Stance

EcoSphere is an ERP-style ESG management platform. Any future AI capability should be advisory and should support operational records rather than replace the main product workflow.
